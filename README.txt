Adds the Cortex tracking code to article pages.

== Description ==

We understand that we aren’t all developers, and this plugin helps you avoid having to dive into your site’s code. Just install the Cortex plugin for WordPress and enable tracking in Settings.
This plugin allows us to use site tracking to create and submit campaigns, organize, display, and filter your data, and to monitor and test domain health through Cortex — all without touching any code.
Simply activate the plugin, enable tracking, and experience the full spectrum of real-time tracking features that Cortex has to offer.


== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Enable tracking in the plugin settings.
